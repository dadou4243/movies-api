About the project

Tools and Libraries

  • The Angular CLI is a great way to generate a new Angular project along with adding new components, modules and services into the application.

  • The PrimeNG UI library has been used to build the movies table component. It offers several built-in features such as columns sorting and filtering, row expansion, scroller and more.

API

All the movies data are requested and received from The Movie Database API.

The API is free and offers a large number of methods. The documentation is very well written and structured.

Architecture

The project architecture is rather simple. The Angular CLI already offers a boilerplate folder structure when the project is initially generated using the ng new command.

For this project, there is no need to create several module, and therefore, all the components and utility modules are imported to the main AppModule.
